# 入队指南

## 入队方式
入队流程：参加新生培训 → 参与考核及选拔 → 最终审核 → 成功入队

### 参加新生培训
参与新生培训并参与考核赛即视为拥有入队资格。

项目部通常在每学年春季学期开始时进行新生培训。培训为线下培训，具体内容视具体安排而定。

### 考核及选拔
新生培训结束后数天内将举行入队考核。考核具体内容贴合新生培训内容，结合培训日常表现进行综合评分，最终择优录取。

### 入队及转正
成功通过考核后即成为预备队员，预备队员结束春季学期的学习后转为正式队员。

## 队员权利
- 拥有实验室固定座位
- 可使用实验室内无线网络、电源
- 可使用实验室内空调、打印机
- 可借阅实验室内的图书资源
