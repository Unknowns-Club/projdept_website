# 2024年新生培训须知

欢迎各位同学参加东北农业大学 ACM 集训队项目部 2024 年纳新暨新人培训。以下是针对本次纳新与培训的详细说明，请你认真阅读。

## 项目部是做什么的？
ACM集训队项目部侧重于学习软件项目开发知识，从课内理论知识出发，提高实际软件开发能力。
经过项目部的培训，你可以学习到常见的应用软件（Web网页、小程序等）的开发方法，从而帮助你参加校内的各项竞赛（如SIPT、互联网+等），你也有机会参与校内软件项目的开发，为你的简历增色，让你更好地应对求职过程中的挑战。

## 在项目部我能学习到什么？
现今的主流Web应用开发分为前端、后端两个方向。项目部的培训包括前端开发和后端开发内容，在学习后，你可以独立开发Web网页、小程序等应用软件。
- 前端开发：主要关注用户可以直接看到且可以直接交互的软件界面部分。主要学习 HTML、CSS、JavaScript 等内容，后续继续学习 Vue.js 等现代框架知识以及网络交互内容。
- 后端开发：主要关注用户不可见但举足轻重的幕后部分，为前端界面提供数据与服务，处理业务逻辑。主要学习 Go 语言、Gin 框架、API 设计与开发，MySQL 和 Redis 数据库等相关工具。

## 培训信息
本次培训并非面向零基础同学。建议至少学习过一门编程语言，有一定的编程基础知识，望各位同学知晓。
- 培训周期：本次培训自 2024 年 3 月 10 日(周日)开始，至选拔赛结束(时间待定)，预计持续一个月时间
- 培训方式：线下培训，分两批次培训
- 批次信息：批次一：每周日/周二/周四，每周一/周三/周五。两批课程由同一课程负责人授课，保证内容基本一致，请根据自身晚课情况选择批次。若有冲突调节不开的，可申请培训请假。每节课课后会提供课程录屏及学习材料，可自行学习。
- 培训地点：电气与信息学院 415 实验室
- 培训时间：(每周日至周五，周六放假) 18:30 至 21:30
- 签到方式：到实验室后扫码签到
- 培训资料：课后会发送每节课的录屏及讲义，便于同学们复习
- 培训内容：本次阶段性培训主题为静态网页开发，属于 Web 前端入门内容。Web 前端开发的后续内容及后端开发将在成为正式队员后继续讲解。

## 在开始学习之前
推荐学习 [Markdown](https://markdown.com.cn/basic-syntax/)，其非常适用于撰写文档，让你无需过多考虑排版问题，可以较好地排版代码。可以用来记笔记，写文档等。

推荐预习 [MDN](https://developer.mozilla.org/zh-CN/docs/Learn) Web 开发教程，这是一套非常完善的 Web 开发入门教程，资料详实又语言有趣，可以作为不错的参考资料。在开始正式学习前，你可以浏览此套教程的开始部分，对 Web 开发有一个大体上的了解，可以为后续学习带来很大的帮助。

遇到问题要善于利用搜索引擎和 GPT 的力量，如果自己难以解决，可以与同学或学长进行交流。

## 结语
在成为预备队员后，你可以按照规定使用集训队提供的资源，如实验室、无线网络、空调及打印机等。请你仔细阅读并遵守《集训队实验室管理制度》(另附)，按时按批次参加培训。

在完成本次阶段性培训的学习之后，你将获得基本的静态网页开发能力，这意味着你已经成功入门 Web 前端开发。但是对于一个完整的应用软件而言，仅仅具备以上知识是不够的。

如果说前端是一个软件的门面，那么后端就是这个软件的核心。Web 后端的开发知识举足轻重，因其相较于 Web 开发相对较难，项目部将其安排在下一阶段的培训当中。不用担心，有了 Web 前端的知识作为基础，学习后端的过程会轻松许多。

如果时间充裕，项目部将在后续继续讲授进阶的 Web 前端开发知识，如 Vue.js，React 框架等，彼时你将掌握构建一个现代化 Web 应用软件的多数基本知识。祝愿你的学习之路一路顺风，也祝愿你能在这个队伍里找到自己所喜爱的方向。
